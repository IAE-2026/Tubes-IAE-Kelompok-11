Pada tugas besar IAE ini, saya berperan sebagai Backend Developer (reservasi service) sekaligus yang mengintegrasikan sistem dengan rekan-rekan saya. Saya membuat backend menggunakan Golang (Gin & GORM) dan mencoba menerapkan Clean Architecture supaya struktur project lebih rapi dan mudah dikembangkan.

Beberapa fitur yang saya kerjakan yaitu Hold Room menggunakan Redis untuk mencegah double booking, fitur Create Booking, serta membuat dokumentasi API menggunakan Swagger.

Selain itu, saya juga mengerjakan integrasi antar microservice menggunakan HTTP Client, menyesuaikan database dan konfigurasi dockernya, serta membantu menyusun konfigurasi Docker Compose dan Nginx API Gateway supaya semua service bisa saling terhubung dan dapat berkomunikasi satu sama lain.

Pada tahap akhir, saya juga ikut menghubungkan aplikasi dengan SSO Cloud, mengimplementasikan SOAP dan RabbitMQ untuk proses komunikasi asynchronus antar layanan.
